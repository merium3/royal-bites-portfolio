# 🍽️ Royal Bite Restaurant Website

A modern, responsive restaurant website developed using HTML, CSS, and JavaScript. The project provides an elegant user experience with multiple pages, interactive features, and a clean design suitable for restaurants and cafés.

---

## 📌 Features

- Responsive design for all screen sizes
- Modern homepage layout
- Interactive navigation menu
- Today's Special section
- Featured Dishes
- Online Order page
- About Us page
- Contact page
- Smooth scrolling
- Clean and organized code
- Mobile-friendly interface

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
